How to set this up on your computer:

Download and install Node.js and MongoDB.
Download these project files to your computer.
Open your Command Prompt in this folder.
Type "npm install" and press Enter. This will download the necessary tools.
Make sure your MongoDB is running.
Type "node server.js" and press Enter to start the server.
Open index.html in your browser.
Now you can save reviews, and they will be stored in your own local database.
